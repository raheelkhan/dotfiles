# Neovim Keybindings

Leader key: `Space`

## File Explorer
| Action | Keybinding |
|--------|------------|
| Toggle explorer | `Space e` |
| Focus explorer | `Space o` |

## Find (Telescope)
| Action | Keybinding |
|--------|------------|
| Find files | `Space ff` |
| Find in files (grep) | `Space fg` |
| Find buffers | `Space fb` |
| Find help | `Space fh` |
| Recent files | `Space fr` |
| Find word under cursor | `Space fw` |

## Buffers (Tabs)
| Action | Keybinding |
|--------|------------|
| Next buffer | `Tab` |
| Previous buffer | `Shift+Tab` |
| Force close buffer | `Space X` |
| Go to buffer 1-9 | `Space 1-9` |

## File Operations
| Action | Keybinding |
|--------|------------|
| Save | `Space w` |
| Save all | `Space W` |
| Close buffer (smart quit) | `Space q` |
| Quit all | `Space Q` |
| New file | `Space n` |

## Editing
| Action | Keybinding |
|--------|------------|
| Undo | `u` |
| Redo | `U` or `Ctrl+r` |
| Move line down | `Alt+j` |
| Move line up | `Alt+k` |
| Duplicate line down | `Alt+Shift+j` |
| Duplicate line up | `Alt+Shift+k` |
| Comment toggle (line) | `Space /` or `gcc` |
| Comment toggle (selection) | `Space /` or `gc` |
| Indent left (visual) | `<` |
| Indent right (visual) | `>` |
| Select all | `Space a` |

## Search
| Action | Keybinding |
|--------|------------|
| Clear search highlight | `Esc` |
| Search and replace | `Space sr` |

## Windows
| Action | Keybinding |
|--------|------------|
| Split vertical | `Space sv` |
| Split horizontal | `Space sh` |
| Close split | `Space sc` |
| Close other splits | `Space so` |
| Go to left window | `Ctrl+h` |
| Go to right window | `Ctrl+l` |
| Go to lower window | `Ctrl+j` |
| Go to upper window | `Ctrl+k` |
| Increase height | `Ctrl+Up` |
| Decrease height | `Ctrl+Down` |
| Decrease width | `Ctrl+Left` |
| Increase width | `Ctrl+Right` |

## Git
| Action | Keybinding |
|--------|------------|
| Preview hunk | `Space gp` |
| Reset hunk | `Space gr` |
| Blame line | `Space gb` |
| Next git hunk | `]g` |
| Previous git hunk | `[g` |

## LSP (Language Server)
| Action | Keybinding |
|--------|------------|
| Go to definition | `gd` |
| Find references | `gr` |
| Hover info | `K` |
| Signature help (insert mode) | `Ctrl+k` |
| Rename symbol | `Space rn` |
| Code action | `Space ca` |
| Previous diagnostic | `[d` |
| Next diagnostic | `]d` |
| Show diagnostic | `Space d` |

## Quickfix
| Action | Keybinding |
|--------|------------|
| Next quickfix item | `]q` |
| Previous quickfix item | `[q` |
| Close quickfix list | `Space cc` |

## Navigation
| Action | Keybinding |
|--------|------------|
| Jump back | `Ctrl+o` |
| Jump forward | `Ctrl+i` |

## Terminal
| Action | Keybinding |
|--------|------------|
| Toggle terminal | `Ctrl+\`` or `Space t` |
| Exit terminal mode | `Esc Esc` |
| Enter terminal insert mode | `i` |

## Vim Essentials (built-in)
| Action | Keybinding |
|--------|------------|
| Enter insert mode | `i` |
| Exit insert mode | `Esc` |
| Delete line | `dd` |
| Copy line | `yy` |
| Paste | `p` |
| Visual mode | `v` |
| Visual line mode | `V` |
| Go to line start | `0` |
| Go to line end | `$` |
| Go to file start | `gg` |
| Go to file end | `G` |
| Search forward | `/` |
| Search backward | `?` |
| Next search result | `n` |
| Previous search result | `N` |
